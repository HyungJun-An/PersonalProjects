# SpringBoot-Project-Community
스프링부트 + JPA
## 🖥️프로젝트 소개
SeSAC 강의를 참고하여 만든 커뮤니티 사이트 입니다.

## 🕰️개발 기간
-   24.12.07 - 24.12.24

## ⚙️개발 환경
- Java 17
- JDK 17
- FrameWork : SpringBoot(3.4.0)
- Database : Mysql 9.0.1
- ORM : Hibernate

## 📌주요기능
- 로그인
- DB값 검증
- ID, PW 찾기
- 로그인 시 세션 생성
- 회원가입
- 아이디 중복확인
- 닉네임 중복확인
- 회원 정보 수정
- 회원 삭제
- 게시글 생성
- 게시글 조회
- 게시글 수정
- 게시글 삭제
- 게시글 좋아요 기능
- 게시글 목록 페이징 처리
- 제목 내용, 작성자 필터로 게시글 검색
